# NGJ2026 《Life Bound》
《Life Bound》是一款 Windows 地城冒險遊戲。
玩家將在迷宮中探索、尋找鑰匙、躲避敵人並挑戰最終關卡。遊戲的特色是視窗本身會成為地城的一部分，玩家的移動、戰鬥與畫面位置緊密連在一起。

操作方式:
方向鍵：移動
空白鍵:攻擊(按鍵1是遠距離/按鍵2是近戰)
使用滑鼠瞄準

下載後請解壓縮 zip，執行 LifeBound.exe 開始遊戲。
