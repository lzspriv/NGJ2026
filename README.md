# 🎮 《Life Bound》— 2026 NGJ 參賽作品

《Life Bound》是一款 Windows 平台的 2D 地城冒險遊戲，為 **2026 Normal Game Jam (NGJ 3rd)** 的參賽成果作品。

* 👥 **我們隊名是：** 遊戲果醬

---

## 💡 遊戲核心設計理念 (Design Concept)

本次 Game Jam 的核心主題為 **「擴張」** 
我們將此概念融入於遊戲的視野機制中：**「視窗（視線範圍）的大小會與玩家的血量緊密掛勾」**。
當玩家血量越多時，視窗與視野便會隨之**擴張**；反之，當受到傷害、血量變少時，視窗則會隨之縮小。玩家的移動、戰鬥與畫面位置緊密相連，帶給玩家打破第四道牆的獨特操作與生存體驗。

---

## ⚔️ 遊戲玩法與機制 (Gameplay)

* **關卡推進：** 每一關的地圖中皆藏有 **3 把鑰匙**，玩家必須在迷宮中探索並全數收集完畢，方能開啟進入下一關的傳送門。
* **房間挑戰與寶箱：** 地圖上會隨機生成寶箱。當玩家踏入房間並將眼前的敵人全數解決後，即可解鎖寶箱，獲得**獎勵三選一**。
* **強敵與強大 BOSS：每隔 **5 關** 就會面臨一場驚心動魄的 **BOSS 關卡**，必須擊敗當前 BOSS 才可以繼續前進。
* 隨著關卡層數越往後推進，怪物的血量與攻擊能力都會變得越來越強。請大家努力挑戰，看看自己究竟能拔得頭籌闖到第幾關吧！

## 🕹️ 操作方式 (Controls)

* **移動：** 方向鍵 `↑` `↓` `←` `→`
* **瞄準：** 使用滑鼠移動瞄準
* **攻擊：** `空白鍵`（可切換遠距離/近戰模式）
  * *按鍵 1：* 遠距離攻擊
  * *按鍵 2：* 近戰攻擊
* **暫停：** `Esc` 鍵

---

## 🚀 如何開始遊戲 (How to Play)

1. 前往本專案的 [Releases] 頁面下載遊戲壓縮檔（或至 [itch.io 參賽頁面](https://itch.io/jam/normal-game-jam-2026) 下載）
2. 下載後請完整解壓縮 zip 檔案。
3. 雙擊執行 `LifeBound.exe` 即可開始進入地城冒險！
